# Grama-Khata

## Project Overview
Grama-Khata is a digital village administration system that helps manage property records, tax details, and citizen information efficiently.

## Features
- Property registration
- Citizen record management
- Tax management
- User authentication
- Dashboard interface

## Technologies Used
- Python
- Flask
- HTML
- CSS
- JavaScript
- SQLite/MySQL

## Modules
- Admin Module
- User Module
- Property Management
- Tax Management
